Self Hosted Home Lab

The site you're seeing right now is not hosted in a big corporate VPS or other site hosting solution! This site is being served proudly from my humble raspberry pi sitting in my home network. Using nGinx Reverse Proxy, Docker and Proxmox, I'm able to host my Home Assistant instance locally, keeping devices connected locally and privately, as well as setting up a bunch of other services - local media library and streaming, and of course, hosting sites! My main docker compose config will be uploaded to my github soon!
